# Informações Diárias - São Paulo

Site com informações atualizadas diariamente sobre:

## 📊 Conteúdo
- **🌤️ Previsão do Tempo**: Temperatura, chuva e condições meteorológicas para São Paulo
- **🚇 Transporte Público**: Status das linhas de trem e metrô de São Paulo
- **💰 Cotações Financeiras**: Dólar, Euro e Bitcoin em tempo real
- **📝 Lembretes**: Tarefas importantes do dia

## 🚀 Tecnologias
- React 18
- Vite
- Tailwind CSS
- Shadcn/UI
- Lucide Icons

## 📱 Características
- Design responsivo
- Interface moderna
- Atualização em tempo real
- Cores e ícones intuitivos

## 🔄 Atualização
Este site é atualizado automaticamente todos os dias com as informações mais recentes.
