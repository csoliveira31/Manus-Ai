# Informações Diárias - São Paulo

Este repositório contém um site com informações diárias atualizadas para São Paulo, incluindo:

## 📊 Informações Disponíveis

- **🌤️ Previsão do Tempo**: Temperatura, condições climáticas e previsão para o dia
- **🚇 Transporte Público**: Status das linhas de trem e metrô de São Paulo
- **💰 Cotações**: Valores atuais do Dólar (USD), Euro (EUR) e Bitcoin (BTC)
- **📝 Lembrete**: Lista de verificação para preparar as coisas da filha para a escola

## 🚀 Tecnologias Utilizadas

- React.js
- Tailwind CSS
- Vite
- Lucide Icons
- shadcn/ui

## 📅 Atualização

Este site é atualizado diariamente com as informações mais recentes.

**Última atualização**: 04 de julho de 2025

## 🔧 Como Executar

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Execute o projeto: `npm run dev`

## 📱 Funcionalidades

- Design responsivo para desktop e mobile
- Atualização automática do horário
- Interface intuitiva e moderna
- Informações organizadas em cards
- Lista de verificação interativa para lembretes
